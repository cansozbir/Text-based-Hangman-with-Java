# Text-based-Hangman-with-Java
Java ile yaptığım yazı tabanlı adam asmaca oyunum.

### Build
```git clone https://github.com/cansozbir/Text-based-Hangman-with-Java.git```

### Run
```cd Text-based-Hangman-with-Java/```

```java -jar run.jar```
